## Description

Testing NestJS + MongooDB (Mongoose) + GraphQL

## Installation

dont forget to change mongodb string connection in `./src/app.module.ts`

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
